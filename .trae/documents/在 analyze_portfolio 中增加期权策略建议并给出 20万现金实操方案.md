# 目标
- 在现有组合分析中自动生成期权策略建议，覆盖买入认购/认沽、卖出认购/认沽、垂直价差、股票担保（备兑认购）、领式策略（collar）、跨式策略（straddle），并输出到 JSON 与 Telegram 摘要。
- 基于用户现金 200,000 RMB 的资金约束与风险参数，给出到期时间、行权价区间、建议张数/数量的可执行方案。

# 集成点
- `src/workflows/portfolio_workflow.py`：在 `PortfolioAnalysisWorkflow.run` 的“最终建议”阶段新增 `OptionsStrategyEngine.generate()`，产出 `options_recommendations` 并并入保存与摘要。
- `portfolio_agent_orchestration/workflows/main_workflow.py`：在 `format_results_for_telegram` 增加“期权建议”分节；在 `save_results` 的结果结构中加入 `options_recommendations`。
- `src/agents/risk_manager.py`：扩展风险评估，增加组合对冲比率、保证金占用、最大损失估算与 VaR（含期权腿）。
- `src/agents/researcher.py`：在逐股建议中增加 `options_overlay` 字段（如对单只标的的备兑、collar、保护性认沽）。
- `src/bot/telegram_bot.py`：在 `_send_new_portfolio_results` 增加“🧩 期权建议”板块，按策略类型与信心度展示要点。

# 数据需求
- 标的当前价格、历史波动率（HV）与隐含波动率（IV）。现有行情与历史数据管线可复用；若缺失 IV，用 HV 近似并设保守折减。
- 合约列表与合约乘数（例如 50ETF/沪深300ETF 期权，合约单位通常为 10,000 份）。
- 组合持仓与权重，用于判断是否可执行备兑认购或 collar。

# 策略生成逻辑
- 资金分配：以 200,000 RMB 为总预算，设置每策略资金上限与保证金上限（例如单策略≤40% 资金与保证金）。
- 行权价选择：
  - 卖出认沽（现金担保）：选择当前价下方 5–10% 的 OTM；期限以次季月或 3 个月为主。
  - 牛市垂直价差：买 ATM/略 ITM 的认购，同时卖出高于 8–12% 的 OTM 认购；期限 2–4 个月。
  - 保护性认沽：对高权重持仓买入当前价下方 5–10% 的认沽；期限 1–3 个月。
  - 领式策略（collar）：持仓基础上买入保护性认沽并卖出 OTM 认购以降低净成本；期限对齐 2–3 个月。
  - 跨式策略（straddle）：在事件驱动或高波动预期时，买入同到期的 ATM 认购+认沽；期限 1–2 个月，头寸严格限额。
  - 备兑认购：对已持有标的，在当前价上方 5–10% 卖出认购；期限 1–2 个月。
- 数量与风控：
  - 以合约乘数与保证金要求计算可承受张数；设置最大回撤与 VaR 不超过总资金的 8–12%。
  - 设定希腊值与止损/滚动规则：IV 回落≥20% 或价差达到 60–70% 最大收益，提前止盈；亏损达到策略预算的 40–50% 时调整或止损。

# 你的 200,000 RMB 现金策略（示例可直接执行）
- 现金担保卖出认沽（卖 put）— 标的：`510300`（沪深300ETF）
  - 到期：次季月（约 3 个月，如 2026-03）
  - 行权价：当前价下方 8–10%（OTM）
  - 数量：2–3 张（合约单位 10,000 份/张）；预算保证金约 100,000–150,000 RMB
  - 目的：以较低风险赚取权利金，被指派则低位接收 ETF。
- 牛市垂直价差（call spread）— 标的：`510050`（上证50ETF）
  - 到期：次季月（约 3–4 个月，如 2026-03）
  - 买入：ATM/略 ITM 的认购；卖出：高于当前价 8–12% 的 OTM 认购
  - 预算：约 30,000 RMB；根据单组净支出确定 5–10 组的具体张数
  - 目的：受控成本参与上行，最大亏损受限于净支出。
- 保护性认沽（买 put）— 针对你组合中权重最高的个股或 ETF
  - 到期：1–2 个月（如 2025-12）
  - 行权价：当前价下方 5–10%
  - 预算：10,000–20,000 RMB；数量与持仓规模匹配（无持仓则以 `510300` 的名义对冲 1–2 张）
- 领式策略（collar）— 若你持有大权重股票/ETF
  - 买入保护性认沽 + 卖出 OTM 认购（上方 5–10%）
  - 到期：2–3 个月；净成本力求接近 0 或小幅净支出
  - 目的：限制下行同时让出部分上行以降低成本。
- 跨式策略（straddle）— 事件驱动/高波动预期时（谨慎小额）
  - 到期：下月或次月（如 2025-12）
  - 认购+认沽：均为 ATM；数量小于总资金的 10% 所能支持的张数
  - 目的：博弈波动扩大，若无重大事件，优先不执行或快速止损。
- 备兑认购（sell call）— 仅在已持有对应标的时
  - 到期：1–2 个月，行权价：当前价上方 5–10%
  - 数量：与持仓张数对应；用于增强持仓现金流。

# 资金与风险控制
- 保证金占用上限：≤ 75%（含被指派情形的最坏情况预留）。
- 单策略预算：≤ 40% 总资金；跨式与保护性认沽合计 ≤ 20%。
- 止盈/止损与滚动：价差达到 60–70% 最大收益提前了结；权利金亏损达到策略预算 40–50% 及时调整或止损；被指派后评估转为备兑认购。

# 备注
- 上述示例基于中国市场常用 ETF 期权（合约单位通常 10,000 份）。具体张数需结合实时价格与权利金报价计算；系统实现将按当前行情自动化出数。
